## 🧪 Lab Assignments Overview

### **Lab Exercise 1: Logic Gate Design**
Implementation of a Single-Layer Perceptron to learn fundamental Boolean logic.
* **Gates Implemented:** AND, OR, NAND, and NOR.
* **Key Concept:** Demonstrating linear separability in basic logic functions.

### **Lab Exercise 2: Glass Classification (SLP)**
Implementation of a Single-Layer Perceptron for multi-class classification.
* **Dataset:** [UCI Glass Identification](https://archive.ics.uci.edu/ml/datasets/glass+identification)
* **Goal:** Classify types of glass based on chemical compositions using a Softmax activation approach.

### **Lab Exercise 3: Multiple Linear Regression**
Using the Perceptron learning rule with a linear activation function.
* **Objective:** Predict continuous values (Income/Salary) based on multiple input features (Age, Experience).
* **Key Concept:** Understanding how the perceptron architecture adapts to regression tasks.

### **Lab Exercise 4: Multilayer Perceptron & Backpropagation**
A from-scratch implementation of a Multi-Layer Perceptron (MLP).
* **Dataset:** [Abalone Dataset](https://archive.ics.uci.edu/dataset/1/abalone)
* **Goal:** Predict the age of abalone from physical measurements.
* **Key Concept:** Gradient descent, hidden layers, and the mathematical implementation of backpropagation.
